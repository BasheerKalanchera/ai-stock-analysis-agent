Context Synthesis

The Qualitative Analyst doesn't work in isolation. It actively consumes outputs from the Strategy (Moat) and Risk (Fraud) agents to ground its "Scuttlebutt" investigation.

Sequential Logic

The Qualitative Pipeline runs Strictly Sequentially to manage API rate limits and ensure logical dependency
(e.g., QoQ needs analysis of both transcripts as input).

Zero-Footprint

Data flows from Screener.in to the UI without ever touching the disk. All data resides in Virtual Memory (RAM) via io.BytesIO.
