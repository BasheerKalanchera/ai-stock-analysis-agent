Trigger Phase

The user initiates a query via the Streamlit UI. This interaction triggers the LangGraph workflow, passing control to the downloader.

Stateless Handoff

Data is kept in RAM (BytesIO). The Screener Agent updates the shared State variable, which the Quant Agent immediately consumes.

Insight Return

Once analysis is complete, control returns to the UI. Streamlit parses the resulting JSON/Dataframes to render the Equity Research Workbench.
